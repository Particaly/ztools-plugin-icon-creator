# 图标编辑器使用指南

## 快速开始

### 创建画布
1. 打开编辑器后会自动创建一个默认画布
2. 点击顶部菜单「文件 → 新建文档」可创建新画布
3. 在属性面板可设置画布预设尺寸或自定义宽高

### 添加对象
- **添加形状**：点击顶部菜单「插入」，选择基础形状（矩形、圆形、三角形等）
- **添加文字**：在「插入」面板中选择文字预设，或自定义文字内容
- **导入图片**：点击「文件 → 导入图片」，支持 PNG、JPG、SVG 等格式

## 对象编辑

### 基础变换
- **移动**：选中对象后拖动
- **缩放**：拖动对象四角的控制点
- **旋转**：拖动对象上方的旋转控制点
- **等比缩放**：按住 `Shift` 键拖动角控制点

### 样式设置
在右侧属性面板可调整：
- **填充**：纯色填充或渐变填充（线性/径向）
- **描边**：颜色、宽度、虚线样式
- **透明度**：0-100% 调节
- **圆角**：可编辑路径对象支持圆角设置

### 高级效果

#### 阴影与模糊
1. 在属性面板找到「效果」区域
2. 点击「添加阴影」可添加投影或内阴影
3. 调整阴影的偏移、模糊半径、颜色等参数
4. 开启「模糊」开关可为对象添加模糊效果

#### 渐变填充
1. 在属性面板选择「渐变」填充模式
2. 选择渐变类型：线性渐变或径向渐变
3. 添加、删除或调整渐变色标
4. 调整渐变角度（线性）或中心点（径向）

#### 高级变换
在属性面板「变换」区域：
- **翻转**：点击水平翻转或垂直翻转按钮
- **倾斜**：输入 Skew X/Y 值实现倾斜变换
- **重置**：一键重置所有变换

### 样式复制
1. 选中源对象，按 `Ctrl/Cmd + Shift + C` 复制样式
2. 选中目标对象，按 `Ctrl/Cmd + Shift + V` 粘贴样式
3. 样式包括：填充、描边、渐变、透明度、阴影、模糊等

### 对象锁定
在属性面板点击锁定按钮，可选择：
- **无锁定**：对象可自由编辑
- **锁定位置**：可缩放旋转，但不可移动
- **锁定尺寸**：可移动，但不可缩放旋转
- **完全锁定**：可选中查看，但不可编辑

图层面板中会显示对应的锁定状态图标。

## 图层管理

### 图层操作
- **上移/下移**：调整图层顺序
- **置顶/置底**：快速移到最上层或最下层
- **显示/隐藏**：点击眼睛图标切换可见性
- **锁定/解锁**：点击锁图标切换锁定状态
- **删除**：点击叉图标删除图层

### 图层搜索
在图层面板顶部输入关键词可快速筛选图层。

### 图层状态图标
- **锁图标**：显示对象的锁定模式

## 分组

### 成组
1. 选中多个对象（按住 `Shift` 点击或框选）
2. 按 `Ctrl/Cmd + G` 或点击属性面板「成组」按钮
3. 组对象可整体移动、缩放、旋转

### 解组
1. 选中组对象
2. 按 `Ctrl/Cmd + Shift + G` 或点击「解组」按钮
3. 对象恢复为独立的图层

## 对齐与分布

在属性面板可快速对齐选中的对象：
- **水平对齐**：左对齐、水平居中、右对齐
- **垂直对齐**：顶对齐、垂直居中、底对齐
- **水平分布**：等间距横向排列
- **垂直分布**：等间距纵向排列

## 布尔运算

### 支持的对象
- 矩形、圆形、三角形、多边形等闭合图形
- 可编辑路径对象
- 直线（通过描边轮廓参与运算）

### 操作步骤
1. 选中至少两个可运算对象
2. 点击顶部工具栏的布尔运算按钮
3. 选择运算类型：
   - **并集**：合并为一个对象
   - **交集**：保留重叠部分
   - **差集**：从底层对象减去上层对象
   - **异或**：保留非重叠部分

### 注意事项
- 文字和图片对象不支持布尔运算
- 成组对象需要先解组
- 差集以最底层对象为基准

## 导出

### 导出 PNG
1. 点击「文件 → 导出」
2. 选择 PNG 格式
3. 设置导出尺寸和背景选项
4. 点击「导出」保存文件

### 导出 SVG
1. 点击「文件 → 导出」
2. 选择 SVG 格式
3. 点击「导出」保存矢量文件

### 复制为图片
- `Ctrl/Cmd + Shift + C` 可将选中对象复制为 PNG 到剪贴板
- `Ctrl/Cmd + Shift + V` 可将选中对象复制为 SVG 到剪贴板

## 撤销与重做

- **撤销**：`Ctrl/Cmd + Z`
- **重做**：`Ctrl/Cmd + Shift + Z` 或 `Ctrl/Cmd + Y`
- 编辑器会自动保存历史记录，支持多步撤销

## 工程保存

### 保存工程
1. 点击「文件 → 保存工程」
2. 首次保存会创建新工程文件
3. 后续保存会覆盖同一文件

### 打开工程
1. 点击「文件 → 打开工程」
2. 选择之前保存的 `.json` 工程文件
3. 所有图层、样式、效果都会完整恢复

## 技巧与窍门

### 精确定位
- 在属性面板直接输入坐标值可精确定位对象
- 输入宽高值可精确调整对象尺寸

### 快速复制
- 按住 `Alt/Option` 拖动对象可快速复制
- `Ctrl/Cmd + D` 可原位复制并偏移

### 多选操作
- 按住 `Shift` 点击可添加选择
- 按住 `Shift` 拖动可框选多个对象
- 选中多个对象时可批量修改颜色、描边等样式

### 锁定比例
- 点击宽高输入框之间的锁图标可锁定宽高比
- 锁定后修改宽度或高度会自动调整另一维度

## 常见问题

**Q: 如何取消选择？**  
A: 点击画布空白处或按 `Esc` 键。

**Q: 对象被锁定无法编辑怎么办？**  
A: 在图层面板或属性面板中点击锁定按钮解锁，或选择不同的锁定模式。

**Q: 如何删除不需要的对象？**  
A: 选中对象后按 `Delete` 或 `Backspace` 键，或在图层面板点击删除图标。

**Q: 导出的图片没有透明背景？**  
A: 在导出对话框中取消勾选「包含背景色」选项。

**Q: 如何让对象精确对齐？**  
A: 使用属性面板的对齐按钮，或在属性面板直接输入相同的坐标值。
