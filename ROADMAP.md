# Icon Creator Roadmap

## 目标概述

本轮规划聚焦 4 个方向：

1. **编辑器交互升级**
   - 空格拖动画布
   - Ctrl + 鼠标滚轮缩放
   - 画布对象右键快捷操作
   - 左侧拖拽上图

2. **界面结构重组**
   - 安全区参考线入口升级
   - 右侧面板顺序调整
   - 左侧插入体系重构
   - left-panel 宽度扩展
   - 右下角模式切换入口

3. **资源与预设体系增强**
   - 素材库行为调整
   - Iconify 默认加载优化
   - 色板/渐变预设管理能力补齐

4. **参考线与画布能力完善**
   - 安全区参考线类型切换
   - 参考线透明度调节
   - Canvas / SVG 模式切换

---

## 需求清单映射

1. 安全区参考线改动：编辑工具栏安全区参考线图标按钮，悬浮时弹出 tooltips 选择其他类型
2. [已完成] 右侧预览和检查放到末尾，排序改为：属性、图层、历史、预览、检查
3. 左侧收纳到单独菜单“插入”分类下，包含形状、文字、模板、素材、图标库，点击后以弹窗形式展示
4. 左侧 left-panel 保留，但宽度增加到 270px，菜单排序为：形状、文字、素材、图标库、模板
5. 使用 ztools-ui 的 ContextMenu 实现画布选中对象快捷操作列表
6. 外部导入的素材和图标、新建的图形默认尺寸比例锁定
7. 按住空格键 + 鼠标拖动实现画布可拖动
8. Ctrl + 鼠标滚轮实现画布缩放
9. 色板和渐变预设标题右侧只展示设置按钮，点击后弹出管理弹窗，支持重置、新增、修改、删除、排序；色板支持分类和列数控制，渐变支持条数控制
10. 素材库仅展示素材列表，保存选中对象到素材库的功能移到 ContextMenu，无数据时展示空状态
11. 参考线支持透明度调节
12. canvas-frame 右下角增加模式切换，可切换 canvas 模式和 svg 模式
13. Iconify 默认按顺序加载图标，滚动到底后继续加载，不再依赖关键词搜索；默认选中全部图标集并支持沉底加载更多
14. 左侧菜单支持拖拽到画布直接上图

## Milestone 1：基础交互与布局重组

**目标：优先完成最直观、最影响编辑流畅度的能力。**

### 1.1 安全区参考线入口升级
- 工具栏中的“Keyline 与安全区参考线”按钮从单一开关升级为可选类型入口
- 悬浮弹出参考线类型选择面板
- 支持模板切换：`none / material / ios / favicon / custom`
- 新增参考线透明度配置

**预期产出：**
- 更清晰的参考线切换入口
- 不同平台参考线模板切换能力
- 透明度即时生效

### 1.2 右侧面板顺序调整 [已完成]
目标顺序调整为：
- 属性
- 图层
- 历史
- 预览
- 检查

### 1.3 left-panel 保留并扩宽
- 左侧面板宽度调整为 `270px`
- 菜单顺序调整为：
  - 形状
  - 文字
  - 素材
  - 图标库
  - 模板

### 1.4 空格拖动画布
- 按住空格进入临时平移模式
- 鼠标拖动画布区域时平移视区
- 松开空格恢复正常编辑模式
- 需要避免与输入框、对象拖拽、点编辑模式冲突

### 1.5 Ctrl + 鼠标滚轮缩放
- 在画布区域支持 `Ctrl + wheel` 缩放
- 缩放范围沿用现有 zoom 规则
- 优先考虑以鼠标位置为中心缩放
- 需要阻止默认页面缩放行为

### 1.6 参考线透明度调节
- 参考线透明度配置对边框、安全区、中心轴线统一生效
- 配置变更后实时刷新 overlay

## Milestone 2：插入体系与对象快捷操作

**目标：完成“插入”入口统一化与对象操作提效。**

### 2.1 顶部“插入”分类弹窗
- 顶部主菜单新增“插入”分类
- 点击后弹出统一插入面板
- 内容包含：
  - 形状
  - 文字
  - 模板
  - 素材
  - 图标库
- 与左侧 left-panel 共存：
  - 顶部“插入”提供统一入口
  - left-panel 提供高频常驻入口

### 2.2 画布对象 ContextMenu
使用 `ztools-ui` 的 `ContextMenu` 为画布选中对象提供快捷操作。

**建议菜单项：**
- 复制
- 删除
- 锁定 / 解锁
- 上移 / 下移
- 置顶 / 置底
- 编组 / 解组
- 保存到素材库
- 可选：重命名、导出选中对象

### 2.3 保存选中对象到素材库迁移
- 移除素材库页顶部“保存选中”按钮
- 将“保存选中对象到素材库”改为 ContextMenu 操作项
- 强化对象操作路径一致性

### 2.4 左侧菜单支持拖拽上图
- 左侧插入项不仅支持点击插入，也支持直接拖拽到画布
- 拖拽落点决定插入位置
- 支持对象类型：
  - 形状
  - 文字
  - 模板
  - 素材
  - 图标库项（可分阶段实现）

---

## Milestone 3：资源导入与素材库体验优化

**目标：完善资源导入后的默认行为与素材工作流。**

### 3.1 默认尺寸比例锁定
以下对象默认锁定宽高比例：
- 外部导入素材
- Iconify 图标
- 新建基础图形

**要求：**
- 默认锁定
- 用户仍可在属性面板手动解锁

### 3.2 素材库仅保留列表展示
- 素材页只负责展示素材列表
- 无数据时提供明确空状态提示
- 空状态文案应引导用户通过对象右键菜单保存素材

### 3.3 素材空状态优化
建议空状态包含：
- 空状态图标 / 插图
- 提示文案
- 操作引导：选中对象后右键可保存到素材库

---

## Milestone 4：Iconify 浏览体验升级

**目标：从“搜索驱动”升级为“浏览 + 搜索并存”。**

### 4.1 默认图标浏览
- 默认加载全部图标集的首批结果
- 不再要求必须输入关键字才能看到内容
- 默认按顺序加载图标

### 4.2 懒加载与沉底加载更多
- 滚动到底部自动继续加载
- 页面底部保留“加载更多”按钮作为兜底
- 支持持续分页浏览

### 4.3 搜索与浏览共存
- 默认进入图标库即展示内容
- 搜索用于缩小范围，而不是作为唯一入口
- 默认图标集过滤为“全部”

---

## Milestone 5：色板与渐变预设管理体系

**目标：将预设能力从“可使用”升级为“可管理”。**

### 5.1 标题区改造
- 色板标题右侧仅保留“设置”按钮
- 渐变预设标题右侧仅保留“设置”按钮

### 5.2 预设管理弹窗
点击设置后弹出统一管理弹窗，支持：
- 重置为系统预设
- 新增
- 修改
- 删除
- 排序

### 5.3 色板增强能力
- 支持分类管理
- 支持控制展示列数

### 5.4 渐变增强能力
- 支持控制展示条数
- 支持新增/修改/删除/排序

### 5.5 数据持久化
- 用户预设与展示配置写入本地存储
- 保留系统预设作为重置基线

---

## Milestone 6：模式切换与高级能力

**目标：补齐更专业的视图与输出模式能力。**

### 6.1 canvas / svg 模式切换
- 在 `canvas-frame` 右下角增加模式切换器
- 支持：
  - Canvas 模式
  - SVG 模式

### 6.2 建议分阶段实现
**V1：**
- 先实现模式切换 UI
- SVG 模式先提供只读预览

**V2：**
- 若后续需要，再扩展到 SVG 代码编辑与双向同步
