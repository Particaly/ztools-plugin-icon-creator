# Icon Creator 远期规划事项文档任务计划书

> 本文档用于记录 Icon Creator 作为“专业图标编辑器”的长期目标、阶段拆解、任务清单和完成记录。
>
> **维护规则：每完成一个任务，必须同步更新本文档**，至少修改任务状态、完成日期、实际产出和后续影响。

## 1. 文档维护约定

### 1.1 状态说明

| 状态 | 含义 |
| --- | --- |
| `未开始` | 已规划但尚未进入开发 |
| `进行中` | 正在设计、编码或验证 |
| `已完成` | 已实现并通过基本验证 |
| `延期` | 暂缓处理，等待前置条件或重新评估 |
| `取消` | 不再计划实现 |

### 1.2 优先级说明

| 优先级 | 含义 |
| --- | --- |
| `P0` | 工具核心闭环能力，优先实现 |
| `P1` | 明显提升图标生产效率和可用性的能力 |
| `P2` | 长期增强能力，优先服务编辑体验、规范辅助和批量交付 |

### 1.3 每次任务完成后的更新要求

完成任一任务后，需要同步更新：

1. 将任务状态改为 `已完成`。
2. 填写完成日期。
3. 在任务的“完成记录”中写明：
   - 主要改动文件
   - 实现范围
   - 验证方式
   - 遗留问题
4. 如任务影响后续规划，需要调整相关任务描述、优先级或依赖关系。
5. 在“阶段进度总览”中更新完成数量。

## 2. 当前产品基线

### 2.1 已具备能力

当前项目已经具备以下基础编辑能力：

- 画布新建、尺寸预设、背景色设置、缩放查看
- 基础形状、文字预设、图片导入
- 对象移动、缩放、旋转、填充、描边、透明度、锁定、删除
- 图层列表、排序、显示/隐藏、锁定/解锁、搜索、重命名
- 成组、解组
- 撤销、重做
- PNG / SVG 导出
- 基于 PathKit 的矢量布尔运算
- 点位 / 线段 / 曲线编辑
- 渐变填充编辑
- 标尺与坐标提示
- 编辑器快捷键查看和自定义
- 万花筒复制效果
- 像素网格显示、网格间距设置、移动 / 缩放 / 点位编辑吸附到网格
- Material / iOS / Favicon / 自定义安全区参考线
- 图标规范检查面板，提示安全区越界、小尺寸描边、颜色数量和非整数边界
- 描边转轮廓，将可见描边转换为可编辑填充路径

### 2.2 主要短板

当前项目已经具备较完整的轻量矢量编辑基础，但作为长期使用的图标编辑器，还需要补齐以下能力：

1. 工程保存、打开、自动保存与可持续编辑能力
2. SVG / Iconify / 模板等图标素材来源
3. 多尺寸、多格式、规范化导出能力
4. 图标设计规范辅助，如网格、安全区、小尺寸预览
5. 更完整的矢量编辑能力，如描边转轮廓、文字转轮廓、图片转矢量探索
6. 图标集合管理与批量交付能力

## 4. 阶段一：补齐图标编辑核心闭环（P0）

阶段目标：让用户可以创建、保存、再次编辑、导入外部 SVG，并按常见图标交付规格导出。

### T1.1 工程文件保存

- **优先级**：P0
- **状态**：已完成
- **完成日期**：2026-06-08
- **目标**：支持将当前画布、图层、对象元数据、画布设置保存为可再次编辑的工程文件。
- **建议拆分**：
  - 定义工程文件 JSON schema
  - 复用 Fabric `toObject` 序列化能力
  - 保留自定义对象元数据，如渐变、编辑路径、万花筒、端点吸附等
  - 新增“保存工程”入口
  - Preload 增加写入工程文件能力
- **验收标准**：
  - 保存后生成 `.iconcreator.json` 或同类工程文件
  - 工程文件包含画布尺寸、背景色、对象列表、图层顺序和自定义元数据
  - 保存后的文件可被后续打开任务正确读取
- **完成记录**：
  - **主要改动文件**：`src/views/home/index.vue`
  - **实现范围**：新增顶部“保存工程”入口，基于 `schemaVersion: 1` 输出 Icon Creator 工程 JSON；复用 Fabric `toObject(SERIALIZED_OBJECT_PROPS)` 保存对象，并额外记录画布宽高、背景色和 `editorObjectId` 图层顺序；复用现有 `window.services.writeTextFile` 生成 `.iconcreator.json` 文件。
  - **验证方式**：执行 `npm run build` 通过。
  - **遗留问题**：当前仍使用下载目录时间戳文件名和浏览器弹窗反馈，后续 T1.6 / T2.9 可继续优化保存路径、toast 和文件名交互。
