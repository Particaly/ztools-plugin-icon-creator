# Home Editor 回归检查清单

> 用途：T4.9 收口 Shell 层后，用于每轮编辑器核心改动的 smoke checklist。自动构建只能覆盖类型与打包，交互类能力仍需按本清单做手工抽查。

## 自动检查

- [x] `npm run build`：类型检查与生产构建通过。

## 手工 smoke checklist

- [ ] 启动编辑器后 Fabric 画布正常初始化，背景、缩放和标尺显示正常。
- [ ] 新建文档、打开 `.iconcreator.json`、保存工程、草稿恢复流程正常。
- [ ] 新增 / 切换 / 重命名 / 删除画板后，画布内容和历史记录不串板。
- [ ] SVG 文件、粘贴 SVG / path、图片、Iconify、模板和用户素材导入后可选中、可撤销。
- [ ] SVG、PNG、批量画板 ZIP 导出，以及复制为 SVG / PNG 正常。
- [ ] 选区变化后属性面板、图层面板、预览面板和检查面板同步正常。
- [ ] 图层搜索、拖拽排序、右键菜单、显示 / 锁定 / 删除 / 重命名正常。
- [ ] 撤销 / 重做覆盖对象新增、删除、移动、样式调整、导入导出前后的关键状态。
- [ ] 点编辑、线段编辑、框选点、端点吸附、箭头参数和曲线控制点行为保持一致。
- [ ] 布尔运算、描边转轮廓、文字转轮廓、图片转矢量主路径正常。
- [ ] 万花筒启用 / 关闭、中心点拖拽、实例同步、脱离源对象正常。
- [ ] 快捷键抽屉、快捷键触发、空格拖动画布、方向键微移不与输入框冲突。
